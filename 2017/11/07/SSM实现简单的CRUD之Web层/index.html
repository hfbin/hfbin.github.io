CRUD API URL 设计

请求类型 url 说明
DELETE /emp/{ids} 删除某一个员工信息
PUT /emp/{empId} 修改员工信息
GET /emp/{id} 查询员工信息
GET checkuser 用户名是否重复
GET /depts 所有部门
POST /emp 员工保存
POST /emp 所有员工（所携带参数不一样）
